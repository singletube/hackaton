# ADR-010: Управление доступом по ссылке

*   **Статус**: Принято
*   **Дата**: 2026-04-17
*   **Контекст**: Требуется возможность делиться файлами через контекстное меню файлового менеджера, не переходя в веб-интерфейс.

## Решение
Реализовать вызовы API облачных провайдеров для генерации и отзыва публичных ссылок. Добавить соответствующие пункты в расширения файловых менеджеров (Nautilus, Caja). Сгенерированная ссылка должна автоматически копироваться в системный буфер обмена.

## Обоснование
1. **Удобство**: Пользователю не нужно открывать веб-версию для того, чтобы поделиться файлом.
2. **Нативность**: Интеграция в стандартное контекстное меню делает функцию интуитивно понятной.
3. **API**: И Яндекс.Диск, и NextCloud поддерживают управление публичными ссылками через свои API.

## Последствия
- Положительные: Выполнение требования задачи по управлению доступом.
- Отрицательные: Необходимость реализации дополнительных API-вызовов для каждого провайдера и взаимодействия с буфером обмена ОС.
